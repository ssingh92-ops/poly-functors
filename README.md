# Polynomial Functors (Containers) in Lean 4
This project formalizes the category Poly (implemented as `PolyC`) of polynomial functors using container semantics in Lean 4, following Niu & Spivak’s *Polynomial Functors: A Mathematical Theory of Interaction*. A polynomial is represented by shapes/positions `(A : Type, B : A → Type)` with semantics `P(X) = Σ a, (B a → X)`, and morphisms are dependent lenses mapping shapes forward and pulling back positions. Current progress includes the core container definition, morphisms, a category instance, an evaluation functor in `X`, verified sum/product/container-composition constructors, and a translation to/from Mathlib’s `PFunctor`; next steps add explicit semantics as a functor into endofunctors, concrete examples (identity/constant/list containers), and a clearer bridge from lenses to natural transformations.
