# EE598 Final Project — Polynomial Functors (Containers) in Lean 4

This project formalizes the category **Poly** of polynomial functors using **container semantics** in Lean 4, following the perspective of Niu & Spivak’s *Polynomial Functors: A Mathematical Theory of Interaction*. Concretely, I represent a polynomial by a type of **positions** and a dependent family of **directions** at each position, interpret it as the endofunctor `X ↦ Σ a, (Dir a → X)`, define morphisms as **dependent lenses** (the “forward on positions / backward on directions” decomposition), and prove the category laws (identity/associativity). Time permitting, I will add constructions like sums/products and the composition product (monoidal structure) and relate lenses to natural transformations.
